# Paper-Rock-Scissors - Java Console Application

## Overview

This is an implementation of a classical Rock-Paper-Scissors game. The user plays against the computer. Number of rounds can be specified.

## Prerequisites

*Java 25* or higher and *Maven* installed.

## Testing Libraries

*JUnit 5 (Jupiter)* and *Mockito*

## How to Build and Run

**Compile and Run Tests:**

   ```bash
   mvn clean test
   ```

**Package the Application:**

   ```bash
   mvn clean package
   ```

**Run the Game:**

   ```bash
   java -jar target/paper-rock-scissors-1.0.0-SNAPSHOT.jar
   ```
